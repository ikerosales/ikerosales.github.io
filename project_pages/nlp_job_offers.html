Project Overview

This project started with a simple question: Can we predict job salaries from job descriptions using NLP? By analyzing thousands of real-world job postings, we built a regression model capable of estimating the expected maximum salary.

The NLP-Data-Job-Offers repository explores the analysis of job offers using Natural Language Processing (NLP) to uncover insights on salary predictions, job-related skills, and categories within job descriptions.

Key Insights

We explored techniques like TF-IDF, FastText, and LDA, unveiling hidden job categories and uncovering the impact of job benefits vs technical skills. Additionally, we tackled the challenge of salary estimation where the data is often unclear or missing.

Among the main findings are:

- Identification of patterns in job descriptions that correlate with specific salary ranges
- Automatic categorization of job offers based on description content
- Analysis of the relative impact of technical skills versus job benefits in salary determination

Technologies Used

This project uses various natural language processing and machine learning technologies and techniques:

Python Pandas NLTK Scikit-learn TF-IDF FastText LDA Regression Data Visualization
